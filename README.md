# ZomaLens - Customer Feedback Sentiment Analysis Tool

A premium sentiment analysis dashboard for Zomato customer feedback, built with React, Vite, Recharts, and Framer Motion.

![ZomaLens Dashboard](placeholder.png)

## Features

- **Dashboard**: Real-time visualization of sentiment trends, rating distribution, and critical issues.
- **Review Management**: Sort, filter, and search through customer reviews.
- **Sentiment Analysis**: Automatic classification of feedback into Positive, Neutral, and Negative categories.
- **Premium UI**: Modern Glassmorphism design with dark mode and smooth animations.

## Tech Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Variables, Glassmorphism)
- **Charts**: [Recharts](https://recharts.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)

### Installation

1.  Clone the repository or navigate to the project directory.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App

Start the development server:
```bash
npm run dev
```

Open your browser to the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure

- `src/components`: UI Components (Dashboard, Sidebar, ReviewList).
- `src/data`: Mock data generation.
- `src/index.css`: Global styles and design system variables.
- `src/App.jsx`: Main application layout.
